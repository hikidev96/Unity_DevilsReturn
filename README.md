# 프로젝트 커밋 규칙

---

### 접두사

- Import new asset : 새로운 에셋을 임포트했을때
- Add new file: 새로운 파일을 추가했을때
- Add new feature : 새로운 기능을 추가했을때
- Update feature, feature name, : 기능을 업데이트 했을때
